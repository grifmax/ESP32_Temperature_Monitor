# ESP32 Temperature Monitor

Это проект для мониторинга температуры с использованием микроконтроллера ESP32, датчика DS18B20 и OLED-дисплея. Устройство может подключаться к Wi-Fi и отслеживать текущую температуру, отображать её на экране, а также отправлять уведомления через Telegram и MQTT.

## Функции проекта
- Измерение температуры с использованием датчика DS18B20.
- Отображение температуры на OLED-дисплее.
- Уведомления через Telegram и MQTT при изменении температуры или достижении пороговых значений.
- Веб-интерфейс для настройки устройства (часы, время работы экрана и т.д.).
- Режимы работы: активный, ожидание, стабилизация температуры.

## Используемое оборудование
- **Микроконтроллер**: ESP32
- **Датчик температуры**: DS18B20
- **Экран**: OLED 0.42" I2C
- **Пищалка**: для уведомлений
- **Батарея**: 18650 через TP4056 (с мониторингом уровня заряда через GPIO 0)

## Установка и настройка
1. Клонируйте этот репозиторий:
    ```bash
    git clone https://github.com/yourusername/esp32-temperature-monitor.git
    ```
2. Откройте проект в PlatformIO или Arduino IDE.
3. Подключите ESP32 к компьютеру и загрузите прошивку.
4. Настройте Wi-Fi и другие параметры через веб-интерфейс или Telegram.
